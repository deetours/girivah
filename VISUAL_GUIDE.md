## 🎨 Design System

### **Color Usage**
```
Primary (Black #0B0B0B)
├── Text
├── Backgrounds
└── Primary buttons

Accent (Orange #FF6A00)
├── CTA buttons
├── Links
├── Highlights
└── Important elements

Backgrounds
├── White (#FFFFFF) - Main
├── Light gray (#F7F7F7) - Secondary sections
└── Dark (#0B0B0B) - Dark sections

Borders & Dividers
└── Light gray (#E5E5E5)
```

### **Typography Hierarchy**
```
H1 - 64px (Hero headlines)
H2 - 48px (Section titles)
H3 - 36px (Subsections)
Body - 18px (Main text)
Small - 14px (Captions)
```

### **Spacing Rhythm**
```
Desktop: 120px between sections
Tablet: 80px between sections
Mobile: 60px between sections
8px grid system for smaller elements
```

---

## 🎬 Animation & Interactions

### **Hover Effects**
- Buttons: Scale up 1.05, add shadow
- Cards: Lift up -8px, border color change
- Links: Color change to orange
- Images: Scale up 1.02 on card hover

### **Page Transitions**
- Fade in on load (300ms)
- Smooth scroll behavior
- No jarring jumps

### **Scroll Effects**
- Hero gradient parallax on scroll
- Smooth section reveals
- No heavy animations

---

## 📱 Mobile Breakdowns

### **Mobile (< 640px)**
- Single column layout for grids
- Full-width sections
- Hamburger menu
- Larger touch targets (44px+)
- No hover effects (tap instead)

### **Tablet (640px - 1024px)**
- 2-column grids (where applicable)
- Reduced padding
- Readable but compact

### **Desktop (> 1024px)**
- 3-4 column grids
- Full design with all spacing
- Hover effects enabled
- Sidebar navigation (where applicable)

## 🎯 Key Visual Principles

1. **Whitespace** - Generous spacing feels premium
2. **Typography** - Large, bold headlines draw eyes
3. **Color** - Orange accent guides attention
4. **Imagery** - Cinematic photos inspire adventure
5. **Consistency** - Repeating patterns create rhythm
6. **Hierarchy** - Clear visual flow guides users
7. **Simplicity** - Minimal chrome, maximum content
